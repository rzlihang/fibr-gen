# fibr-gen

**fibr-gen** 是一个基于 Excel 模板的灵活报表生成器。

项目名称 **fibr-gen** 取自 "**f**LEX**ib**LE **r**EPORT **gen**ERATOR" 的字母组合。

## 简介

本系统旨在根据指定的模板和配置生成 Excel 报表。它通过**数据视图（Data View）**将数据源抽象化，并通过**数据块（Block）**将数据映射到 Excel 单元格中。

## 主要特性

- **基于模板**：使用标准的 Excel (`.xlsx`) 文件设计报表样式。
- **数据视图**：通过 SQL 或内存视图将数据获取与报表布局解耦。
- **灵活的数据块**：
  - **值块 (ValueBlock, type: `value`)**：用于简单的数据填充（如列表、单值）。
  - **矩阵块 (MatrixBlock, type: `matrix`)**：支持复杂的布局，如交叉表（透视表），支持水平和垂直方向的动态扩展。
  - **表头块 (HeaderBlock, type: `header`)**：定义矩阵布局的表头或轴。
- **动态工作表**：根据数据动态生成多个工作表（例如按部门或月份分表）。
