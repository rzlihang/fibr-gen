# fibr-gen

**fibr-gen** は、Excelテンプレートに基づいた柔軟なレポート生成ツールです。

プロジェクト名 **fibr-gen** は、"**f**LEX**ib**LE **r**EPORT **gen**ERATOR"（柔軟なレポート生成器）に由来しています。

## 概要

本システムは、指定されたテンプレートと設定に基づいてExcelレポートを生成します。**データビュー（Data View）**を使用してデータソースを抽象化し、**ブロック（Block）**を使用してデータをExcelセルにマッピングします。

## 主な機能

- **テンプレートベース**: 標準的な Excel (`.xlsx`) ファイルを使用してレポートをデザインできます。
- **データビュー**: SQLまたはメモリベースのビューを使用して、データ取得とレポートレイアウトを分離します。
- **柔軟なブロック**:
  - **値ブロック (ValueBlock, type: `value`)**: 単純なデータ入力（リスト、単一値）に使用します。
  - **マトリックスブロック (MatrixBlock, type: `matrix`)**: クロス集計（ピボットテーブル）のような複雑なレイアウトをサポートし、垂直および水平方向への動的な拡張が可能です。
  - **ヘッダーブロック (HeaderBlock, type: `header`)**: マトリックスレイアウトのヘッダーまたは軸を定義します。
- **動的シート**: データに基づいて複数のシートを動的に生成します（例：部門別や月別のシート作成）。
